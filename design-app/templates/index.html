O que é?

Este é um sistema automatizado de verificação e extração de informação de sentenças jusdiciais. Através dele, podemos disponibilizar uma versão amigável de sentenças expedidas pelo Tribubal de Contas de Pernambuco.

Leitura dos dados

Gráficos e charts ajudam a guiar o leitor no entendimento do porquê das decisões e oferecem um melhor noção das quantidades e valores descritos.

Features

O sistema é dinâmico através do uso de tags para que os usuários sigam as partes relevantes do conteúdo das decisões judiciais.
